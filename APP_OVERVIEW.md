# BundledContent (PulseCommand)

## What It Is

BundledContent is a fully automated AI marketing platform that replaces a $10K/month social media agency for small and medium businesses. One subscription gives a business a complete content operation — social posts, short videos, podcasts, newsletters, lead magnets, performance reports, and reputation management — all generated by AI and delivered on autopilot.

Built with Next.js 16, Supabase, and a stack of AI APIs, the platform handles everything from onboarding a new client to publishing 150+ pieces of content per month across every major channel.

---

## How It Works

### For the Customer

1. **Free Demo** — Enter your website, answer 3 questions. The system scans your brand, generates sample social posts, a podcast episode, and a video short. You get an email when it's ready. No signup required.

2. **Sign Up** — Pick a plan (Lite $99/mo, Full $745/mo, or Premium). Go through onboarding: brand asset collection, avatar selection, voice selection, and a phone interview with an AI agent that learns your business.

3. **Monthly Content Delivery** — Every month, the system automatically generates and publishes:
   - **150 social posts** (Instagram, Facebook, LinkedIn, X, Google Business) with AI-generated images
   - **4 short-form videos** (Reels, TikTok, Shorts) with AI avatar presenters
   - **2 podcast episodes** (full 2-host conversational format, distributed to Spotify/Apple)
   - **1 performance report** with engagement metrics and recommendations
   - Optional add-ons: email newsletter, lead magnet PDF, presentation decks, infographics, reputation management

4. **Dashboard** — Customers log in to review content, approve posts, view analytics, manage social connections, and configure add-ons.

### For the Admin

- **Admin Panel** — Manage all clients, trigger content generation, view pipeline status, send emails, manage billing
- **Automated Crons** — Monthly content generation, weekly social posting, demo follow-up emails all run on schedule
- **Pipeline System** — Modular pipeline routes handle each step: generate strategy → create content → render audio → render video → publish posts → deliver to client → send reports

## API Architecture

### Demo Pipeline
```
POST /api/demo/check          → Check if email already has a demo
POST /api/demo/start           → Create demo request, fire generate
POST /api/demo/generate        → Scan website, generate posts + podcast + video
GET  /api/demo/status/[id]     → Poll for completion
POST /api/demo/verify          → Email verification
POST /api/demo/render-video    → Trigger video rendering
```

### Onboarding
```
POST /api/auth/sign-up                    → Create account + client record
GET  /api/auth/me                         → Current user data + onboarding step
POST /api/onboarding/scan-website         → AI website analysis (Claude + Jina)
POST /api/onboarding/save-brand-assets    → Save brand data
POST /api/onboarding/save-selection       → Save avatar/voice choice
POST /api/onboarding/send-interview-reminder → SMS reminder
POST /api/onboarding/complete             → Finalize onboarding
```

### Content Pipeline
```
POST /api/pipeline/monthly             → Orchestrate full monthly generation
POST /api/pipeline/generate-strategy   → Create brand strategy
POST /api/pipeline/generate-content    → Generate social post text + images
POST /api/pipeline/render-audio        → Create podcast via AutoContent
POST /api/pipeline/submit-heygen       → Create video short via AutoContent
POST /api/pipeline/publish-posts       → Push to Ayrshare
POST /api/pipeline/deliver-client      → Notify client content is ready
POST /api/pipeline/reports             → Generate performance report
POST /api/pipeline/generate-lead-magnet → Create PDF lead magnet
POST /api/pipeline/send-newsletter     → Generate and send email newsletter
POST /api/pipeline/generate-brief      → Create strategy brief
POST /api/pipeline/create-audiogram    → Create audio visualization
POST /api/pipeline/deploy-page         → Deploy AI landing page
POST /api/pipeline/analyze-transcript  → Analyze interview transcript
```

### Cron Jobs
```
/api/cron/monthly-generation   → Trigger content generation for all active clients
/api/cron/monthly-delivery     → Deliver generated content to clients
/api/cron/weekly-social        → Publish scheduled social posts
/api/cron/demo-followup        → Send follow-up emails to demo leads
```

### Social Publishing
```
POST /api/ayrshare/init        → Create Ayrshare profile for client
GET  /api/ayrshare/connect     → Generate JWT link for social account linking
POST /api/ayrshare/publish     → Publish content to connected platforms
POST /api/ayrshare/disconnect  → Remove social connection
```

### Integrations
```
GET  /api/integrations/google-reviews          → Fetch Google reviews
GET  /api/integrations/yelp-reviews            → Fetch Yelp reviews
POST /api/integrations/reputation/setup        → Configure review monitoring
POST /api/integrations/reputation/draft-response → AI-draft review response
POST /api/integrations/reputation/submit-response → Post review response
```
